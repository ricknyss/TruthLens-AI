# TruthLens AI

## AI-Powered Digital Media Authenticity & Image Forensics Platform

TruthLens AI is an advanced AI-powered platform designed to detect image manipulation, deepfakes, and AI-generated content. The system helps users verify the authenticity of digital media through forensic analysis, explainable AI, and trust scoring.

---

## Problem Statement

With the rapid growth of AI-generated content and image editing technologies, it has become increasingly difficult to distinguish authentic images from manipulated ones. TruthLens AI aims to provide a reliable solution for detecting image tampering, deepfakes, AI-generated media, and other forms of digital forgery using advanced Computer Vision and Deep Learning techniques.

---

## Features

- Image Tampering Detection
- Object Removal Detection
- Object Insertion Detection
- Copy-Move Forgery Detection
- Image Splicing Detection
- Deepfake Detection
- AI-Generated Image Detection
- Error Level Analysis (ELA)
- Metadata Analysis
- Explainable AI (Grad-CAM)
- Tampered Region Localization
- Confidence Score Generation
- Forensic Report Generation
- Scan History Management
- Android Mobile Application
